Array.at과 Array[idx]의 차이

Posted on Aug 2, 2024

인트로

Array.at(idx)은 es2022부터 등장한 메소드이다.

기존 방식의 Array[idx]과 어떤 차이가 있는지 알아보자.

차이

가장 극명한 차이는 음수 인덱스 지원에 있다.

const arr = [10, 20, 30]
console.log(arr[-1]) // undefined

const arr = [10, 20, 30]
console.log(arr.at(-1)) // 30
